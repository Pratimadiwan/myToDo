# Todo List 2024

A simple and interactive Todo List application built with HTML, CSS, and JavaScript, featuring local storage for persistent task management.

## Features

- **Add Tasks:** Easily add new tasks to your list.
- **Mark as Complete:** Check off completed tasks.
- **Delete Tasks:** Remove individual tasks or clear the entire list.
- **Filter Tasks:** View all tasks, pending tasks, or completed tasks.
- **Local Storage:** Your tasks are automatically saved locally, so you can pick up where you left off.
- **Task Counts:** Track the total number of tasks and the number of completed tasks.
- **User-Friendly Interface:** Clean and intuitive design for an optimal user experience.

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. **Clone the Repository:** `git clone https://github.com/your-username/todo-list-2024.git`
2. **Open `index.html` in your browser.**

## Project Structure

```
todo-list-2024/
├── index.html
├── style.css
└── index.js
```

- `index.html`: Contains the HTML structure of the Todo List.
- `style.css`: Styles the appearance of the Todo List.
- `index.js`: Provides the JavaScript functionality for task management, local storage, and user interactions.

## Future Enhancements

- **Due Dates:** Add functionality to set deadlines for tasks.
- **Task Prioritization:** Allow users to prioritize tasks by importance.
- **Categorization:** Enable task grouping by categories or projects.
- **Cloud Syncing:** Implement cloud storage for cross-device synchronization.
- **Responsive Design:** Ensure optimal viewing and interaction on all screen sizes.
